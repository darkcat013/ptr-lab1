# Real Time Programming, Laboratory work nr. 1

Implemented using Erlang

## Install erlang on windows with chocolatey

```
$ choco install erlang
```

## Build

Example for week1

```
$ cd week1
$ erl
$> c(week1).
```

## Run function

Example for week1

```
$> week1:hello_world().
```

## Run test

Example for week1

```
$> week1:test().
```
